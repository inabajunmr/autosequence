# HTTP Request Sequence Recorder

HTTPリクエストを記録してMermaidシーケンス図を生成するChrome拡張機能です。

## 機能

- ボタンクリックでHTTPリクエストの記録開始/停止
- 同じドメインを同じparticipantとしてグループ化
- Mermaidシーケンス図の自動生成
- SVGエクスポート機能
- Mermaidコードのクリップボードコピー

## インストール方法

1. Chromeで `chrome://extensions/` を開く
2. 「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 使用方法

1. 拡張機能のアイコンをクリックしてポップアップを開く
2. 「開始」ボタンでHTTPリクエストの記録を開始
3. ブラウザで通常通りウェブサイトを閲覧
4. 「停止」ボタンで記録を終了
5. 「図を表示」でMermaidシーケンス図を表示

## ファイル構成

- `manifest.json` - 拡張機能の設定
- `popup.html/js` - ポップアップUI
- `background.js` - バックグラウンドスクリプト（HTTPリクエスト監視）
- `diagram.html` - シーケンス図表示ページ

## 注意事項

- HTTPSサイトでのみ動作します
- 大量のリクエストがある場合、パフォーマンスに影響する可能性があります
- ダイアグラムは最大50リクエストまで表示されます
